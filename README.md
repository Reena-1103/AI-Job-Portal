An AI-powered Job Portal developed using Python, Django, HTML, CSS, and SQLite.

- User Registration & Login
- Job Search
- Apply Jobs
- My Applications
- Dashboard
- AI Career Recommendation
- Admin Panel

## Technologies

- Python
- Django
- HTML
- CSS
- SQLite

1. Clone Repository
