# 🤖 AI Job Portal

An AI-powered Job Portal developed using **Python**, **Django**, **HTML**, **CSS**, and **SQLite**. The application enables users to register, securely log in, search for jobs, apply for jobs, track applications, and receive AI-based career recommendations through a simple and user-friendly interface.

# 📖 Project Description

The AI Job Portal is a web-based application designed to simplify the recruitment process by connecting job seekers with available job opportunities.

The system allows users to:

- Register an account
- Securely log in
- Search available jobs
- Apply for jobs
- Track submitted applications
- Receive AI-based career recommendations based on their skills

An administrator can manage users, jobs, applications, and recommendations using the Django Admin Panel.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Reena-1103/AI-Job-Portal.git
```

Go to the project directory:

```bash
cd AI-Job-Portal/backend
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Django server:

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
